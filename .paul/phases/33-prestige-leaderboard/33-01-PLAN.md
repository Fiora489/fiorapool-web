    Define a scoring scheme that rewards breadth + depth:
    - +100 per unlocked title
    - +5 per app level
    - +10 per longest win streak game (capped at 20 streak)
    - +50 per badge chain fully completed (from BADGE_DEFS)
    - +1 per 10 lifetime wins

    Build `computePrestigeScore(matches, progress, unlockedTitleIds)`:
    - Compute each component, return { total, breakdown: [{label, value, points}] }
    - For titleRanks: sort titles by threshold difficulty (proxy: use threshold numeric value relative to typical achievement cost). Tag each with rarity label:
      • rank 1–4: Common (bronze)
      • rank 5–8: Rare (silver)
      • rank 9–11: Epic (purple)
      • rank 12–14: Legendary (gold)

    Rarity ordering is based on approximate difficulty — use threshold alone isn't enough (different units); hard-code a difficulty map:
    rookie 1, veteran 5, iron_man 6, centurion 7, aram_devotee 7, polymath 8, vision_lord 9, carry 10,
    streak_demon 10, comeback_king 11, mechanical 12, grandmaster 13, one_trick 13, marathoner 14
    (higher = harder).

    Returns `titleRanks` array sorted by difficulty ascending.
