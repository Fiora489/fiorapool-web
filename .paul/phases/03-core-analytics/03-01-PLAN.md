## Goal
Build the analytics data layer — a typed computation module and `/api/analytics/[type]` route — and wire up an analytics overview page visible in the nav.

This plan delivers the full data layer so 03-02 and 03-03 are pure UI work.

    --- Types to export ---

    export type OverviewStats = {
      total: number
      winRate: number
      avgKills: number
      avgDeaths: number
      avgAssists: number
      avgCsMin: number
      topChampions: { name: string; games: number; wins: number; winRate: number }[]
      queueBreakdown: { type: string; games: number; wins: number; winRate: number }[]
      recentForm: boolean[]
    }

    export type AramStats = {
      total: number
      winRate: number
      avgKills: number
      avgDeaths: number
      avgAssists: number
      avgDamagePerMin: number
    }

    export type ClutchStats = {
      clutchWins: number
      totalWins: number
      clutchRate: number
      examples: { champion: string | null; duration: number; goldDeficit: number | null }[]
    }

    export type OpponentQualityStats = {
      opponents: { name: string; games: number; wins: number; winRate: number }[]
    }

    export type TeamCompStats = {
      archetypeCounts: Record<string, number>
      mostCommonDuo: string
    }

    --- Computation functions to export ---

    export function computeOverview(matches: MatchRow[]): OverviewStats
      - Refactor the logic already in src/app/api/analytics/route.ts into this function
      - topChampions: top 5 by games
      - recentForm: last 10 matches win/loss booleans

    export function computeAram(matches: MatchRow[]): AramStats
      - Filter: matches.filter(m => m.queue_type === 'ARAM')
      - avgDamagePerMin: avg of (damage_dealt / (game_duration_seconds / 60))
      - If no ARAM matches: return { total: 0, winRate: 0, avgKills: 0, avgDeaths: 0, avgAssists: 0, avgDamagePerMin: 0 }

    export function computeClutch(matches: MatchRow[]): ClutchStats
      - A win is "clutch" if: m.win === true AND (m.gold_diff_at_10 !== null && m.gold_diff_at_10 < 0)
        OR m.win === true AND m.game_duration_seconds > 28 * 60
      - clutchWins: count of clutch wins
      - totalWins: total wins
      - clutchRate: Math.round((clutchWins / totalWins) * 100) or 0 if no wins
      - examples: up to 5 clutch win rows, map to { champion: m.champion_name, duration: m.game_duration_seconds, goldDeficit: m.gold_diff_at_10 }

    export function computeOpponentQuality(matches: MatchRow[]): OpponentQualityStats
      - Group by enemy_champion_name (skip rows where enemy_champion_name is null)
      - Sort by games descending
      - Return all opponents (no limit — UI can paginate)

    export function computeTeamComp(matches: MatchRow[]): TeamCompStats
      - Use the user's champion_name column (not raw_data — team comp from user perspective only for now;
        full 10-player team comp deferred to when raw_data parsing is added)
      - Apply CHAMPION_ARCHETYPES lookup map (hardcoded, ~60 common champions)
      - Count archetype occurrences across all matches
      - mostCommonDuo: top 2 archetypes joined with '+' (e.g. "Fighter+Mage")

    CHAMPION_ARCHETYPES — define as a Record<string, string> constant in the file.
    Include at minimum these champions (add more as you see fit):
      Tank: Malphite, Ornn, Leona, Nautilus, Alistar, Blitzcrank, Thresh (support-tank), Rammus, Shen, Galio
      Assassin: Zed, Talon, Katarina, Akali, Fizz, Ekko, Rengar, Kha'Zix, Qiyana, Yone (split — add to Fighter if preferred)
      Mage: Lux, Syndra, Viktor, Orianna, Azir, Cassiopeia, Veigar, Xerath, Zoe, Heimerdinger, Brand, Twisted Fate
      Marksman: Jinx, Caitlyn, Ezreal, Jhin, Vayne, Tristana, Kai'Sa, Aphelios, Xayah, Miss Fortune, Ashe, Sivir
      Support: Soraka, Janna, Lulu, Nami, Yuumi, Sona, Karma, Bard, Zilean, Seraphine, Zyra, Morgana
      Fighter: Darius, Garen, Irelia, Fiora, Camille, Renekton, Aatrox, Jax, Tryndamere, Wukong, Mordekaiser, Olaf

    Fallback: any champion not in the map → 'Unknown'
  </action>

       - For 'opponent-quality': select `win,enemy_champion_name`

    5. Dispatches to correct function from analytics.ts:
       'overview'         → computeOverview(matches)
       'aram'             → computeAram(matches)
       'clutch'           → computeClutch(matches)
       'opponent-quality' → computeOpponentQuality(matches)
       'team-comp'        → computeTeamComp(matches)
    6. Returns 200 with result JSON
