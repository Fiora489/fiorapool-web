
  <action>
    --- src/lib/analytics.ts ---

    Add a new exported type after TeamCompStats:

    export type VisionObjectivesStats = {
      total: number
      avgVisionScore: number
      avgWardsPlaced: number
      avgWardsKilled: number
      avgVisionPerMin: number
      topVisionGames: { champion: string | null; visionScore: number; durationSeconds: number }[]
    }

    Add a new exported function after computeTeamComp:

    export function computeVisionObjectives(matches: MatchRow[]): VisionObjectivesStats
      - total = matches.length
      - If total === 0: return all zeros, topVisionGames: []
      - avgVisionScore: mean of vision_score across all matches
      - avgWardsPlaced: mean of wards_placed
      - avgWardsKilled: mean of wards_killed
      - avgVisionPerMin: mean of (vision_score / (game_duration_seconds / 60))
        Guard: skip if game_duration_seconds === 0
      - topVisionGames: matches sorted by vision_score descending, take top 5,
        map to { champion: m.champion_name, visionScore: m.vision_score, durationSeconds: m.game_duration_seconds }
      - Round avgVisionScore, avgWardsPlaced, avgWardsKilled to 1 decimal; avgVisionPerMin to 1 decimal

    --- src/app/api/analytics/[type]/route.ts ---

    1. Add `computeVisionObjectives` to the import from '@/lib/analytics'
    2. Add `'vision-objectives'` to the VALID_TYPES array
    3. Add case to the switch:
       case 'vision-objectives': return NextResponse.json(computeVisionObjectives(rows))

    IMPORTANT: Update the AnalyticsType union and VALID_TYPES array — must stay in sync.
    The switch must remain exhaustive after the new type is added.
