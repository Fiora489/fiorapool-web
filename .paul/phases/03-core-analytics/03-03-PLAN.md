## AC-1: Clutch factor page renders
```gherkin
Given an authenticated user with won matches lasting over 28 minutes
When they navigate to /analytics/clutch
Then the page shows: clutch rate %, clutch win count, total win count,
  and a list of up to 5 clutch examples (champion, duration, label)

Given a user with no wins meeting the clutch threshold
When they navigate to /analytics/clutch
Then the page shows "No clutch wins yet" with a back link to /analytics
```

## AC-2: Opponent quality page renders
```gherkin
Given an authenticated user with synced matches that have enemy_champion_name data
When they navigate to /analytics/opponent-quality

  each showing: champion name, games count, wins count, win rate %

Given a user with no opponent data (enemy_champion_name always null)
When they navigate to /analytics/opponent-quality
Then the page shows "No opponent data yet" with a back link to /analytics
```
