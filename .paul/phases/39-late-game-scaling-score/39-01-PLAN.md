    Duration buckets:
    - 'short': game_duration_seconds < 25 * 60
    - 'mid':   25*60 ≤ game_duration_seconds ≤ 35*60
    - 'long':  game_duration_seconds > 35 * 60

    Per-bucket: games, wins, winRate.

    Delta = longWr - shortWr (signed). Score formula:
    - delta <= -20 → 0 (early-game crusher)
    - delta == 0  → 50 (balanced)
    - delta >= +20 → 100 (late-game monster)
    - Linear interp, clamp 0-100.

    Tier:
    - 0-24 Early Game Crusher
    - 25-49 Early Leaning
    - 50-74 Balanced
    - 75+ Late Game Monster

    Champions scaling: for each champion with ≥5 games, compute short/long split (any split; include mid-only if no short/long).
    Score per champion = same delta formula. Sort by games desc, top 8.
    Label each champion 'early' | 'balanced' | 'late' based on their score.
