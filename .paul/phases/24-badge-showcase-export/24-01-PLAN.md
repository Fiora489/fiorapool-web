## Goal
Build `/export/badges` — a dedicated page with 3 badge-card layouts (Grid / Highlight / Chains), live preview, and PNG download. Mirrors the stats-card export pattern from Phase 23.

## Purpose
Players want to flex their badge collection on social — earned badges grid, a featured "highlight" card, or chain-progress overview.
</objective>


```gherkin
Given user visits /export/badges
When page loads
Then 3 layout buttons render, current selection shows live preview <img>,

```

## AC-3: All 3 Layouts Render Without Error
```gherkin
Given any of the 3 layouts is selected
When the API is called
Then a 800×400 PNG is returned with layout-specific content
```
