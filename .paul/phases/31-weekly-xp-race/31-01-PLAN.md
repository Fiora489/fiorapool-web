  <files>src/lib/weekly-xp.ts</files>
  <action>
    New module:
    - `weekStart(date)` — return Monday ISO date string (YYYY-MM-DD) for the week containing the date
    - `formatWeekLabel(isoStart)` — "Mon DD" of isoStart for compact display
    - `WeekBucket` type
    - `computeWeeklyXp(matches)`:
      Walk matches chronologically, tracking running win streak. For each match, compute XP:
        - win: xpForWin(streakLength) where streakLength is current streak BEFORE this win
          (so first win has streak=0, second consecutive win has streak=1, etc.)
        - loss: xpForLoss() = 30; reset streak
      Group XP into weekly buckets keyed by Monday-of-captured_at.
      Return last 8 weeks (oldest first). Pad missing weeks with zero buckets.
      Set thisWeek = bucket whose isoStart === weekStart(today); lastWeek = bucket before that;
      bestWeek = max-XP bucket across all matches (not just last 8 — could be an older record).
      daysRemaining = 7 - dayOfWeek(today, Monday=0)

    Use lib/xp.ts xpForWin and xpForLoss.
  </action>
