<objective>
## Goal
Build `/export/stats-card` — a page with Hero / Scoreboard / Timeline layout selector, live preview, and PNG download. Extends the existing `/api/share/card` route to accept `?layout=hero|scoreboard|timeline` and render different layouts.

## Purpose
Currently only the Hero layout exists at the API route, with no UI to choose or preview. Players want a shareable stats card and Twitter/Discord post — they need to see what they're sharing first.
</objective>
